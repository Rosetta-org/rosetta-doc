Errors
Error logging with automatic bug assignment and fewer notifications. Opbeat integrates with Git so we can cross-reference errors with releases, and pull the code that is breaking into the stacktrace.

Get started for free

Beautiful, annotated stacktraces

We’ve put lots of thought into making our stacktraces as helpful as possible, and have brought together all the information you need.

  • See the full commit information, including who wrote the code that is breaking for reduced fingerpointyness.
  • See the code directly in the stacktrace with syntax highlighting for easy scanning.
  • Access the full stacktrace from your cellphone, so you can determine the severity of the error when you’re AFK.

Auto-assignments

When an error occurs Opbeat automatically assigns it to the developer best placed to fix the error, taking into account who wrote the code and when it was deployed. Dramatically reducing debugging time, cutting noise, and improving workflow.

Opbeat is the only service that has the ability to do this.

It's not about blame

To err is human and every developer ships errors. Our non-judgemental algorithm automatically assigns the bug, so you can skip finger pointing, stay friends with your coworkers, and focus on coding.

Better together

Metrics

See how response times are affected by errors with our simple metrics, for a clearer picture of how the issue is evolving.

Releases

Full integration with release tracking lets you see exactly who changed each line of code and when those lines were put into production.

Workflow

Opbeat cross-references errors, releases and metrics for a smarter, faster and simpler workflow.

Less notification noise

You’ll receive fewer but more relevant notifications, so you can spend more time being productive.

Smart grouping

Similar errors are aggregated intelligently, letting you review, resolve & ignore related errors together.

Smart notifications

Choose to be notified if a new error occurs once, or many times in succession.

Relevant data

See local variables & filter out frames from external libraries. Attach custom data to any error.

Error archive

Has a bug come back from the dead? We archive the data so regressions can be traced.

Snooze or silence alarms

Ignore an error for two hours while you fix it, and silence non-pertinent errors permanently.

Sort by

See the most occurrences, newest group or last seen at the top of your feed.

Task queues

Built in support for Delayed::Job, Celery, Resque, and Sidekiq.

Getting started

Firing up Opbeat error logging in your Django project is easy, simply install it with pip. You can also read our full Django documentation here.

$ pip install opbeat
